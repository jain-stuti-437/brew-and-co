# ☕ Brew & Co.

## 🌐 Live Demo

🔗 https://jain-stuti-437.github.io/brew-and-co/

A responsive, fairy-inspired café website built using **HTML** and **CSS** as part of my front-end web development journey. This project focuses on creating an immersive user experience with elegant layouts, smooth animations, and responsive design.

## ✨ Features

* 🧚 Fairy-themed aesthetic with a warm coffeehouse atmosphere
* 📱 Fully responsive design for desktop, tablet, and mobile devices
* 🎨 Modern UI designed using HTML and CSS
* ☕ Interactive menu section
* 📅 Reservation form with styled input fields
* 🖼️ Animated image gallery
* ✨ CSS animations and hover effects
* 🎭 Custom SVG decorative elements
* 📞 Contact section and footer

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox
* Media Queries
* CSS Animations
* SVG Graphics
